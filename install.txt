Simple Install
1. Login to WordPress and go to Plugins > Add New > Upload
2. Upload restrict-content-pro-braintree.zip and activate
3. Go to Restrict Content Pro settings and configure

Manual Install
1. Upload the restrict-content-pro-braintree folder to your /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Restrict Content Pro settings and configure
